# Audio Files

Place your wedding music files in this directory.

## Supported Formats
- MP3 (recommended)
- OGG
- WAV

## File Naming
- `wedding-music.mp3` - Main background music
- `wedding-music.ogg` - Fallback format

## Music Suggestions
- Classical wedding music
- Soft instrumental pieces
- Christian hymns
- Romantic ballads

## File Size
Keep audio files under 5MB for optimal loading performance.

## Copyright
Ensure you have the rights to use any music files you add to this project.
